# Assignment 1 - Multithreaded File Processing Application

## Objective

Develop a multithreaded file processing application using Java. The application demonstrates file handling, multithreading, and logging concepts.

## Features

* Read files using `BufferedReader`
* Write files using `BufferedWriter`
* Process multiple files simultaneously using threads
* Log processing results to a log file
* Use clean coding standards and object-oriented design

## Technologies Used

* Java
* Multithreading
* BufferedReader
* BufferedWriter
* File Handling
* Exception Handling

## Project Structure

```text
Assignment1_MultithreadingFileHandling
│
├── Main.java
├── FileProcessor.java
├── FileProcessingTask.java
├── LoggerUtil.java
├── input1.txt
├── input2.txt
├── output1.txt
├── output2.txt
├── processing.log
└── README.md
```

## Setup Instructions

### Prerequisites

* Java JDK 8 or later installed
* VS Code, IntelliJ IDEA, Eclipse, or any Java IDE

### Step 1: Clone the Repository

```bash
git clone <repository-url>
```

### Step 2: Navigate to the Project Directory

```bash
cd Assignment1_MultithreadingFileHandling
```

### Step 3: Compile the Java Files

```bash
javac *.java
```

### Step 4: Run the Application

```bash
java Main
```

## Input Files

### input1.txt

```text
hello world
java programming
multithreading example
```

### input2.txt

```text
file processing
using buffered reader
and writer
```

## Expected Output

The application creates:

* output1.txt
* output2.txt
* processing.log

### Sample Console Output

```text
All files processed successfully.
```

### Sample Log Entry

```text
2026-06-02T10:00:10 - Thread-1 processed input1.txt | Lines: 3
2026-06-02T10:00:10 - Thread-2 processed input2.txt | Lines: 3
```

## Learning Outcomes

* Understanding Java Multithreading
* Working with Threads and Runnable Interface
* Reading files using BufferedReader
* Writing files using BufferedWriter
* Logging application activities
* Exception Handling in Java
