Assignment 1 - Library Management System
📌 Description

This project is a Library Management System developed using Java programming language.
It applies Object-Oriented Programming (OOP) concepts along with Collections Framework to manage books and users efficiently.

The system allows users to perform basic library operations such as adding books, managing users, borrowing and returning books, along with exception handling for invalid operations.

✨ Features
Add new books to the library
Add users to the system
Borrow books from the library
Return borrowed books
Display available books
Exception handling for invalid operations
Usage of Stream API and Lambda Expressions (Java 8 features)
🛠 Technologies Used
Java (Core Java)
Object-Oriented Programming (OOP)
ArrayList
HashMap
Java 8 Features (Streams & Lambda Expressions)
Exception Handling
